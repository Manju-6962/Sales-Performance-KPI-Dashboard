# 📊 Sales Performance Analysis

## 📌 Overview
This project analyzes sales data to understand business performance across categories, regions, and time.

## 🎯 Objectives
- Identify top-performing categories and regions
- Analyze sales trends over time
- Detect profit-impacting factors

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- Excel
- Power BI

## 📂 Dataset
- Sales dataset containing 9,983 records
- Fields: Order Date, Region, Category, Sales, Profit, Quantity

## 🔄 Workflow
1. Data Cleaning (missing values, datatype conversion)
2. Exploratory Data Analysis (EDA)
3. Business Analysis
4. Dashboard creation (Power BI)

## 📊 Key Insights
- Technology category generated highest revenue
- West region is the top-performing region
- Discounts negatively impact profit

## 📈 Dashboard Features
- KPI cards (Sales, Profit, Quantity)
- Sales by Category & Region
- Monthly Sales Trend
- Top Products

## 📌 Conclusion
The analysis helps in identifying growth areas and optimizing pricing strategies.
